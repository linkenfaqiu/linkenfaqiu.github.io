百度网盘突破三界限制达到真仙境的方法

发表于 2021-08-29 更新于 2021-09-19 分类于 技术向 阅读次数：
本文字数： 757 阅读时长 ≈ 1 分钟

厉某一直很想把红色警戒3下载过来玩，找到资源后由于没有百度网盘会员而导致下载速度实在太慢，于是放在网盘积灰许久，后来逛游戏社区max+幸得一份破解百度网盘下载速度限制的秘籍，因此在这里做一个整理，便于日后使用，毕竟天下苦百度网盘久矣！

首先，我们打开百度网盘挑选一个幸运儿下载看看，我选了一部比较好看的电影：《让子弹飞》。

可以看到下载速度保持在100KB/s以下，真有你的昂百度网盘。只需10个小时就能把将近3个G的东西下载过来了，嗯···

虽然说要支持正版，但还是破解比较香···以下开始正题。

第一步：下载Aria2下载器

可以通过极下进行下载，点击网站最上面工具栏的“使用帮助”下载Aria2。

PS：当然也可以在极下官网进行下载，不过极下的只有三个文件，比较便捷。

第二步：解压aria2.zip文件并运行点此启动.bat，从而打开Aria2。

第三步：安装油猴插件Tampermonkey，插件适用于(适用于 Chrome, Microsoft Edge, Safari, Opera Next, 和 Firefox。

第四步：输入https://greasyfork.org/zh-CN，进入油猴脚本下载地址，然后搜索百度网盘，安装图片上第三个脚本。

第五步：在安装油猴插件的浏览器登陆网页版百度网盘。如果脚本安装成功，在进入百度网盘界面后油猴插件会有显示脚本开启，会发现多了一个红色的按钮“简易下载助手”。

第六步：把要下载的东西打勾 -> 点击红色按钮“简易下载助手” -> 点击获取直链地址 -> 点击发送至Aria2 -> 点击下载后可以修改保存路径。

第七步：到Aria2网页中看看，发现下载速度已经到达校园网的最大速度，比百度网盘快多了！

PS:不知道为啥···我明明把文件改过名字了，再下载一次还是原来的名字，当算当算。
